Дана карта букв в виде массива массивов. Ваша задача - определить, можно ли, используя эту карту, построить слово.

Слово может строится только из букв, которые прилегают друг к другу в этой карте. Также эти буквы должны идти в том порядке, в котором они идут в собственно карте

Input: Два аргумента. Первый - это карта букв в виде массива массивов. Второй - это слово, которые мы ищем.

Output: Булеан. Можно ли построить слово используя эту карту.

Example:

  ['E', 'F', 'G', 'H'],

 'BAEF') == true

  ['E', 'F', 'G', 'H'],
  ['I', 'J', 'K', 'L']],
 'ABCDE') == false
